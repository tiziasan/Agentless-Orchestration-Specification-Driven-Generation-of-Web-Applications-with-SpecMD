# Agentless-Orchestration-Specification-Driven-Generation-of-Web-Applications-with-SpecMD

Supplemental material and replication package for the paper *Agentless Orchestration: Specification-Driven Generation of Web Applications with SpecMD*.

This repository contains the SpecMD specification template, the SpecMD Builder tool, the source of every generated web application, the human evaluation material, and the full results for RQ1, RQ2, and RQ3.

## Repository contents

- `SpecMD-tool/` SpecMD Builder, the browser tool that compiles a SpecMD specification into a runnable TypeScript project + example of the SpecMD Markdown specification template.
- `Screenshots/` folder contains the side-by-side captures shown to evaluators, with the SpecMD Builder and WebGen-LM-32B applications presented under neutral labels for each instruction.
- `SpecMD-Generated-Webapp` The 101 web applications generated by SpecMD Builder as a zip file.
- `Results/` Full results for RQ1 (functional benchmark), RQ2 (human evaluation), and RQ3 (code characterization) + The survey instrument used in the human evaluation.

  
## Running the SpecMD Builder tool

SpecMD Builder is a single-file React application that runs entirely in the browser. No build step or server is required.

Open `specmd-builder/index.html` in a web browser.

From there you can author a specification by hand or infer it from a free-text prompt, then generate and download the application as a ZIP.

## Running a generated web application

Each generated application is a standard Vite 5 + TypeScript 5 project. To run one, enter its folder and execute:

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite in your browser.



To replicate the functional test check the methodology on the baseline repository https://github.com/mnluzimu/WebGen-Bench/tree/main
